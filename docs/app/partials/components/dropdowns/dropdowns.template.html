— API: Available functions

Action Description
dropdown() If you add a value (boolean true|false) this function will behave as closeDropdown() when false and as openDropdown() when true. Regular execution of this function is without any parameter; so will toggle the dropdown state. Be aware adding a parameter to this function is not recommended, please use the funtions close and open instead to fullfill your needs. The parameter included mainly serves an internal usage
closeDropdown() Closes the dropdown and updates the state of the element; does nothing if not required
openDropdown() Opens the dropdown and updates the state of the element; does nothing if not required
