The text of the button is determined by the title attribute on the bp-dropdown element. The content is determined by the HTML content inside of the bp-dropdown element. The content can be either a ul as in the above example or whatever you like wrapped in a div. As in default Bootstrap dropdowns, the "dropup" class also works.

If the bp-element is assigned a title attribute, it will wrap that text in a button. If not, it will provide a borderless icon. The icon attribute accepts class names for the the icon, which is a caret by default. Both the title and icon attributes can be changed post-render in any way and the component will update itself approoriately

As a proper Web Component, we've given the bp-dropdown element native javascript methods that can be called on instances of the element to control the dropdown state of that element.

document.getElementsByTagName("bp-dropdown")[0].dropdown(); //toggles dropdown state document.getElementsByTagName("bp-dropdown")[0].openDropdown(); //opens dropdown if closed document.getElementsByTagName("bp-dropdown")[0].closeDropdown(); //closes dropdown if opened

These examples target the first bp-dropdown element on page, but you can substitute 'document' with any instance of any element that contains a bp-dropdown element. This is particularly useful for components/controllers in single page app frameworks.
