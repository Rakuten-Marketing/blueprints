Installation

I just want to include Blueprints CSS stylesheet

You can grab the compiled styles directly from the repository. Then you can include the CSS and fonts into your project. Make sure that fonts directory is publicly accessible via /fonts.

I want to install Blueprints with NPM

The example below references the dev branch of Blueprints repository. You can also reference a tag if you prefer to lock it down to a specific version.

{ //... "dependencies": "git+ssh://git@github.private.linksynergy.com/SD/blueprints.git#dev" }

Then you can include the Blueprints stylesheet like so (given node_modules is publicly accessible).

I want to compile Blueprints myself

To compile Blueprints yourself, it is fairly straight forward as long as you have the right build tools setup, e.g. Gulp, Webpack or Grunt. Blueprints is written in SASS, so you will need node-sass library or a plugin that is specific to the build tool you're using.

/* app.scss */ @import 'node_modules/blueprints/src/blueprints';

I want to override the variables Blueprints provides

Sure, I hear ya! You can have a look at the example below:

/* app.scss */ $rm-white: #F6F6F6; $rm-black: #444444; $rm-primary: #BADA55; $rm-secondary: #884466; @import 'node_modules/blueprints/src/blueprints';
