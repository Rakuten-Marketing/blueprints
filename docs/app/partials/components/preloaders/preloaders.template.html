Reusable loading animations

Having in mind the fact that single page applications can get quite bulky very quickly even when you optimize everything, pushes us to the need of having light and "nice" loading animations or preloaders. Bear in mind there is no way to avoid the waiting times, if you are waiting for response from an API but one option to reduce the frustration of your users is to add a nice custom preloader screen.
