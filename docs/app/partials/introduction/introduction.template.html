What is Blueprints?

An UI framework using Bootstrap 3 as its foundation. Created in that way primarily focusing on look and feel, described as a long living styleguide that can evolve over time and empower developers to build application pages quickly whilst conforming with a consistent branding that is agreed and defined.

Start using this styleguide

You have several options to include the distribution files into your project but the result should be the same: get a copy of the stylesheet file and the javascript into your main HTML document.

There are two recommended options:

Option 1:

Install Blueprints as a node module and compile a unified css from our scss files. You can specify a release version or a branch

// Add this dependency to package.json { "dependencies": "git+ssh://git@github.private.linksynergy.com/SD/blueprints.git#v1.0.0-rc.10" }

Then include dist/blueprints.min.js in your build and in your app's scss file `@import 'node_modules/blueprints/src/blueprints.scss';` (or only include the components you want if you need to optimize). Then in each file where you want to (and you should want to) you can include our size and color variables and helpers/mixins. This will help your custom CSS keep to the same rules and measurement Blueprints applies.

Option 2:

Include the precompiled CSS and JS files from the node module (or bower component) in your app's HTML. This allows you to update versions by just changing the pinned version in your package.json

// Add these to your HTML <link rel="stylesheet" type="text/css" href="node_modules/blueprints/dist/blueprints.min.css"> <script type="text/javascript" src="node_modules/blueprints/dist/blueprints.min.js"></script>

Option 3:

Include compiled CSS and JS from our S3 bucket (https://blueprints-prod.s3.amazonaws.com). Each released version gets its dist folder uploaded to a directory sharing that name on that S3 bucket.

<link rel="stylesheet" type="text/css" href="https://blueprints-prod.s3.amazonaws.com/{insert version here}/blueprints.min.css"> <script type="text/javascript" src="https://blueprints-prod.s3.amazonaws.com/{insert version here}/blueprints.min.js"></script>
