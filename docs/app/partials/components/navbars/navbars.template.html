Navigation bars
Responsive element/layout markup with a webcomponent option

The navbar is a wrapper that positions branding, navigation, and other elements in a single container. It’s easily extensible and can be get super powers if used as a webcomponent, having as the main feature its integrated responsive behavior.

All the following attributes are optional but you can take advantage of them to fit your requirements:

Attribute Description
responsive Boolean: true|false true by default. If present and "true" will also react to the document changes; this because sometimes the used mediaqueries are not enough to handle the responsive behavior in the modern web.
template String: HTML Template reference
undefined by default. Shadow DOM template; if present the menu option will be ignored and the template content will be used instead, just add the name as reference to that template|object
menu String: HTMLElement[classname|id|name]
navbar-center by default. A refence using the clssname, id or name of an element. This element will be used as the area to collapse when the space is not enough to display its content
